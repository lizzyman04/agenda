# Phase 1 Plan 4: L10n Scaffolding Summary

flutter gen-l10n pipeline with ARB files for PT-BR (default) and EN, LocaleCubit persisting locale to SharedPreferences, and a parity test enforcing key sync between both ARB files at every test run.

## What Was Built

### Task 1: ARB files, l10n.yaml, flutter gen-l10n, LocaleCubit (commit 6e632c9)

Created the full l10n scaffolding:

- `lib/config/l10n/l10n.yaml` — configures flutter gen-l10n with PT-BR as template locale, `synthetic-package: false`, output to `lib/generated/l10n/`
- `lib/config/l10n/app_pt_BR.arb` — 6 translation keys in PT-BR (template locale)
- `lib/config/l10n/app_en.arb` — same 6 keys in EN
- `lib/config/l10n/app_pt.arb` — generated by flutter gen-l10n as a by-product of the `pt` base locale detection
- `lib/generated/l10n/app_localizations.dart` — generated `AppLocalizations` class with both locale delegates
- `lib/application/shared/locale/locale_cubit.dart` — `LocaleCubit` with `setLocale()`, PT-BR default, SharedPreferences persistence
- `lib/application/shared/locale/locale_state.dart` — `LocaleState extends Equatable` holding the active `Locale`

ARB keys defined in both PT-BR and EN:
- `appName`
- `settingsTitle`
- `languageLabel`
- `privacyStatement`
- `emptyStateTitle`
- `emptyStateAction`

### Task 2: Wire LocaleCubit into app.dart, ARB parity and bloc tests (commit 450c770)

- `lib/app.dart` updated — `BlocProvider<LocaleCubit>` + `BlocBuilder<LocaleCubit, LocaleState>` drives `MaterialApp.locale`; all four l10n delegates wired (`AppLocalizations.delegate`, `GlobalMaterialLocalizations`, `GlobalWidgetsLocalizations`, `GlobalCupertinoLocalizations`)
- `test/l10n/l10n_test.dart` — 5 tests covering:
  - ARB key parity (set equality between `app_en.arb` and `app_pt_BR.arb`)
  - `LocaleCubit` initial state: PT-BR default when no pref stored
  - `LocaleCubit` initial state: EN when prefs returns `"en"`
  - `setLocale(Locale('en'))` emits `LocaleState(Locale('en'))` and calls `prefs.setString`
  - `setLocale(Locale('fr'))` is a no-op (unsupported locale)

All 5 tests pass. `flutter analyze --no-fatal-infos` exits 0.
